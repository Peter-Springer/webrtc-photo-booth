### WebRTC-Photo-Booth

#### This is a basic photo booth application. A user can enter the number of photos they want to take. Photos are then captured every 5 seconds.

#### This application is built using:
##### 1. Create React App
##### 2. WebRTC
##### 3. Redux
##### 4. React Router
##### 5. Tailwind

#### Running the application:

`npm install`

`npm start`
